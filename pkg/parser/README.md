# Extractor

API для парсера данных аниме из разных источников.

## Навигация по пакетам

- `extractor` - Предоставляет интерфейсы, которые необходимо имплементировать для парсинга из источника.
- `models` - Модели, которые должны возвращать методы экстрактора. Так же используются для резолвинга.
- `runner` - Пакет предоставляющий раннер для одного экстрактора.
